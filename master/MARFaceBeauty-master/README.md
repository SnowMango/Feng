# MARFaceBeauty

一款类似与FaceU的基础版本美颜相机（缩水了贴图功能，其余功能都在）。

*** 

### Feature

* 支持开源算法的美颜（基于GPUImage）
* 支持对焦，闪光灯，前后摄像头转换
* 支持短按拍照，长按摄像
